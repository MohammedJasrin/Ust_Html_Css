Types of tags in HTML

Container Tags which don't have their own content

Example
  • table
  • body
  • head
  • div

Stand alone Tags which need not be closed

Example
  • break
  • image
  • br
  • link

Contente tags which must be opened and closed when the content is entered in between opening and closing tag

Example
  • h1
  • td
  • bold
  • ilatic

Parent and child Tags which will have its own content

Example
  • table: tr,td
  • ul: li
  • ol: li
  • section: article

Tags which are having attributes

Example
  • id
  • class
  • anchar
  • image
